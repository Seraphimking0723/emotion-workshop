我的情緒百寶箱

一、問題背景

在幼兒發展階段，情緒經驗頻繁且強烈， 但孩子缺乏語言與認知能力去理解與表達， 因此情緒常轉化為哭鬧或行為。

同時，家長在面對孩子情緒時， 往往傾向讓情緒「說出來」， 但孩子無法做到， 造成親子溝通落差。

二、議題分析

無法辨識情緒

無法說出需求

情緒轉為行為

情緒引導方式不足

需要用猜的

缺乏表達管道

三、設計思考路徑

孩子不是不願意，而是沒有工具。

設計的核心是讓情緒「被看見」。

情緒 → 角色

混亂 → 分類

哭鬧 → 語言

透過角色與互動建立情緒辨識能力。

四、設計策略

讓情緒可視覺化
透過操作理解情緒
建立情緒語言

五、使用者對應

問題：不會說情緒
解法：角色辨識
問題：不會引導
解法：提供語言
問題：衝突
解法：互動

六、情緒角色設計

愉悅與安全感
失落與低落
強烈情緒
不安與恐懼
排斥與抗拒
挫折與無力感

七、設計價值與反思

將抽象情緒轉化為可理解與可操作的系統， 讓孩子逐步建立表達能力。

並透過親子互動延伸至生活。
